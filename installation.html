Installation¶

The instruction on this page installs the following Inrupt libraries:

Libary

Description

solid-client

Inrupt solid-client is a client library for accessing data stored in Solid Pods.

By default, solid-client only enables access to public data on Solid Pods.

solid-client-authn

Inrupt solid-client-authn is a set of libraries used to authenticate with Solid identity servers.

vocab-common-rdf

Certain agreed-upon Vocabularies exist to identify common data (e.g., name, title, address, url). You can use these common vocabularies to identify the data you want to access. Inrupt’s vocab-common-rdf library bundles some common vocabularies for use in your application.

You can use npm to install solid-client, solid-client-authn, and vocab-common-rdf:

npm install @inrupt/solid-client @inrupt/solid-client-authn-browser @inrupt/vocab-common-rdf
