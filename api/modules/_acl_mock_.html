addMockFallbackAclTo¶

▸ addMockFallbackAclTo‹T›(resource: T): T & WithFallbackAcl

Defined in acl/mock.ts:88

Function for use in unit tests to mock a SolidDataset that has one of its Container’s ACL attached.

Warning: do not use this function in actual production code. This function initialises a new empty ACL and attaches it to a given [[SolidDataset] as its fallback ACL. This is useful to mock a SolidDataset with an ACL in tests of code that call e.g. getFallbackAcl.

since Not released yet.
