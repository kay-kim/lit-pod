getAgentAccess¶

▸ getAgentAccess(resourceInfo: WithAcl & WithResourceInfo, agent: WebId): Access | null

Defined in acl/agent.ts:71

Find out what Access Modes have been granted to a given Agent specifically for a given Resource.

Keep in mind that this function will not tell you what access the given Agent has through other ACL rules, e.g. public or group-specific permissions.

Also, please note that this function is still experimental: its API can change in non-major releases.

Parameters:

Name

Type

Description

resourceInfo

WithAcl & WithResourceInfo

Information about the Resource to which the given Agent may have been granted access.

agent

WebId

WebID of the Agent for which to retrieve what access it has to the Resource.

Returns: Access | null

Which Access Modes have been granted to the Agent specifically for the given SolidDataset, or null if it could not be determined (e.g. because the current user does not have Control Access to a given Resource or its Container).

getAgentDefaultAccess¶

▸ getAgentDefaultAccess(aclDataset: AclDataset, agent: WebId): Access

Defined in acl/agent.ts:222

Given an ACL SolidDataset, find out which access modes it provides to an Agent for the associated Container Resource’s child Resources.

Keep in mind that this function will not tell you:

  • what access the given Agent has through other ACL rules, e.g. public or group-specific permissions.

  • what access the given Agent has to the Container Resource itself (see getAgentResourceAccess for that).

Also, please note that this function is still experimental: its API can change in non-major releases.

Parameters:

Name

Type

Description

aclDataset

AclDataset

The SolidDataset that contains Access-Control List rules for a certain Container.

agent

WebId

WebID of the Agent for which to retrieve what access it has to the Container’s children.

Returns: Access

Which Access Modes have been granted to the Agent specifically for the children of the Container associated with the given ACL SolidDataset.

setAgentDefaultAccess¶

▸ setAgentDefaultAccess(aclDataset: AclDataset, agent: WebId, access: Access): AclDataset & WithChangeLog

Defined in acl/agent.ts:275

Given an ACL SolidDataset, modify the ACL Rules to set specific default Access Modes for a given Agent.

If the given ACL SolidDataset already includes ACL Rules that grant a certain set of default Access Modes to the given Agent, those will be overridden by the given Access Modes.

Keep in mind that this function will not modify:

  • access arbitrary Agents might have been given through other ACL rules, e.g. public or group-specific permissions.

  • what access arbitrary Agents have to the Container itself.

Also, please note that this function is still experimental: its API can change in non-major releases.

access

The Access Modes to grant to the Agent.

Returns: AclDataset & WithChangeLog
