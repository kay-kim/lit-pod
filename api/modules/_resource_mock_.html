mockContainerFrom¶

▸ mockContainerFrom(url: Url | UrlString): Unpromisify‹ReturnType‹typeof getSolidDataset››

Defined in resource/mock.ts:76

Function for use in unit tests to mock a persisted SolidDataset representing a Container.

Warning: do not use this function in actual production code. This function initialises a new SolidDataset but, as opposed to createSolidDataset, it adds metadata that would also have been present had an empty SolidDataset been fetched from the given URL, and was a Container. This is useful to mock a SolidDataset in tests of code that call e.g. isContainer.

since Not released yet.
